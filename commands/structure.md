Прочитай `SKILL.md` скилла `pravo-ritorika` и референсы `references/dispositio.md`, `references/rhetoric-errors.md`.

Отработай запрос пользователя в режиме «Структура»:

1. Определи жанр и адресата текста; если адресат не назван и не выводится — спроси.
2. Составь карту текста: каждая часть опознана (exordium / narratio / propositio / argumentatio / refutatio / peroratio или их сращения), роль названа.
3. Пройди чек-лист разбора структуры из `dispositio.md`: положение тезиса, порядок аргументов (гомеров против порядка проверки), кандидаты на вылет, взаимный подрыв, работа narratio, начало и конец, жанровые требования.
4. Дефекты называй по каталогу `rhetoric-errors.md`, с адресом в тексте.
5. Предложи порядок с обоснованием каждого перемещения. Переписывай текст только по явной просьбе — и тогда меняй порядок и связки, сохраняя формулировки автора.

Формат отчёта — «Формат отчёта по структуре» из SKILL.md.
